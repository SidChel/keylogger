# keylogger

>> <b>keylogger1.2</b><br>
->Store the log text file into the local directory<br>
->to end the program press 'esc' button<br>

>> <b>keylogger1.4</b><br>
->Store the log text file into the local directory<br>
->Store the log text file into a ftp server<br>
