# keylogger
There are two versions of the keylogger

>> <b>keylogger1.2</b>
->Store the log text file into the local directory
->to end the program press 'esc' button

>> <b>keylogger1.4</b>
->Store the log text file into the local directory
->Store the log text file into a ftp server
->to end the program press 'esc' button
